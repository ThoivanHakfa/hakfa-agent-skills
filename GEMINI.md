# GEMINI.md

## User Preferences

- **Language Reference**: Always refer to the language as **Hakfa** (or **Hak-fa**). Avoid "Hakka Chinese", "Taiwanese Hakka", "Hakka dialect", or treating Hakfa as a dialect of Mandarin/Chinese.
- **Terminology**: Use the term **Roman Orthography** (or "roman orthography") instead of "Romanization" or "romanization".
- **Orthography Names**:
  - Hakfa Latin-script Orthography
  - Hakfa Roman-script Orthography
  - Use **PFS (Pha̍k-fa-sṳ)** for the 長老教會 Presbyterian tradition.
  - Use **KPPY (Kàu-pō͘ Phin-yîm / 教育部客家語拼音方案)** for the official MOE standard.
  - **Avoid:** "Hakka Romanization", "Hakka Pinyin", "MOE Pinyin", or conflating PFS with KPPY — they are distinct systems.
- **Tone Numbering**: Use traditional 長老教會 PFS tone numbers **1–6** (Si-yen 四縣腔). Do not repurpose POJ tone numbers 1–8.

## Agent Skills

This repository contains packaged `.skill` files for Gemini CLI. If you need to perform Hakfa orthography conversions or apply terminology rules, ensure you have the relevant `.skill` installed via the `gemini skills install` command and reloaded.
